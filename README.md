# CHEESEHub

WAX blockchain DeFi & NFT platform powered by the $CHEESE token.

## Features

- Token swaps via Alcor DEX
- NFT drops (CHEESEDrop) via NFTHive
- DAO governance
- NFT farming
- Token locking
- PowerUp resource management
- CHEESE Null burning
- Banner advertising
- Drip token distribution

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- WharfKit (WAX blockchain)
- TanStack Query

## Development

```sh
bun install
bun run dev
```

## Deployment

Deployed to GitHub Pages via GitHub Actions on push to `main`.
